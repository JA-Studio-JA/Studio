# JA Studio GitHub Pages leht

See on staatiline GitHub Pagesi näidisleht domeenile `ja-studio.ee`.

## Kontakt
Praegu on kontaktiks lisatud:
`hello-ja.studio@outlook.com`

## Failid
- `index.html` – avalehe sisu
- `style.css` – kujundus
- `CNAME` – domeen GitHub Pagesi jaoks
- `assets/` – SVG dekoratsioonid ja ikoonid

## GitHub Pages seadistus
1. Loo GitHubis uus repository, näiteks `ja-studio`.
2. Laadi kõik failid repository juurkausta.
3. Ava `Settings` → `Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Custom domain: `ja-studio.ee`.
7. Domeeni DNS-is suuna `ja-studio.ee` GitHub Pagesi peale.

## Märkus
Kui hiljem teed Tally vormi, asenda `mailto:` lingid Tally vormi lingiga.


## Päris piltide lisamine

Asenda `assets` kaustas allolevad failid oma päris piltidega:

- `logo.svg` – sinu JA Studio logo SVG kujul
- `hero.jpg` – suur avalehe hero-pilt
- `kevad.jpg` – Kevade kollektsiooni pilt
- `suvi.jpg` – Suve kollektsiooni pilt
- `sugis.jpg` – Sügise kollektsiooni pilt
- `talv.jpg` – Talve kollektsiooni pilt
- `kutsekomplekt.jpg`
- `paeva-paberitooted.jpg`
- `taiskomplekt.jpg`

Soovituslik pildi suurus:
- hero.jpg: vähemalt 1600 × 1000 px
- kollektsioonikaardid: vähemalt 900 × 650 px
- komplektikaardid: vähemalt 900 × 500 px

Failinimed peavad täpselt samaks jääma.
