# JA Studio GitHub Pages leht

See on staatiline GitHub Pagesi näidisleht domeenile `ja-studio.ee`.

## Kontakt
Praegu on kontaktiks lisatud:
`hello-ja.studio@outlook.com`

## Failid
- `index.html` – avalehe sisu
- `style.css` – kujundus
- `CNAME` – domeen GitHub Pagesi jaoks
- `assets/` – SVG dekoratsioonid ja ikoonid

## GitHub Pages seadistus
1. Loo GitHubis uus repository, näiteks `ja-studio`.
2. Laadi kõik failid repository juurkausta.
3. Ava `Settings` → `Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Custom domain: `ja-studio.ee`.
7. Domeeni DNS-is suuna `ja-studio.ee` GitHub Pagesi peale.
