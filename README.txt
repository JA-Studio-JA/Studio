JA Studio – GitHub Pages failid

Laadi need failid GitHubi repository juurkausta:

- index.html
- style.css
- CNAME
- assets/ja-studio-logo.svg

Oluline:
Ära laadi GitHubi üles ainult ZIP-faili. ZIP tuleb enne arvutis lahti pakkida ja siis failid/kaustad repository juurkausta üles laadida.

Kui GitHub küsib, kas olemasolevad failid üle kirjutada, vali kinnitamine/commit.
