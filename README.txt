JA Studio – töötav ZIP pakett

Failid:
- index.html
- CNAME
- README.txt

Kõige olulisem:
1. Paki ZIP arvutis lahti.
2. ÄRA laadi GitHubi ZIP-faili ennast.
3. Laadi GitHubi repository juurkausta üles index.html ja CNAME.
4. index.html peab olema otse repo esimesel tasemel, mitte kausta sees.
5. CNAME faili sees peab olema ainult: ja-studio.ee

See versioon ei vaja style.css ega assets kausta, sest kogu kujundus on index.html faili sees.
Pärast commit'i oota 1-3 minutit ja tee brauseris Ctrl + F5.
